This project is a monolithic PHP web application that implements a simple task management system with user authentication. The application follows a classic client–server workflow where the browser sends HTTP requests to PHP scripts running on the server. These scripts handle input validation, session management, and business logic, and interact with a MySQL database using prepared statements to securely store and retrieve data. Users can register, log in, and manage their personal tasks by creating, updating, deleting, and marking them as finished or pending. The frontend is built with HTML, CSS, and JavaScript, while the backend is handled entirely in PHP
